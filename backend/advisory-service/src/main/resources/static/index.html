Recommended sequence
Submit assessment → copy assessmentId → generate report → copy reportId → fetch report.

Submit Assessment

Create a new assessment request

POST /assessments

Get Assessment

Fetch an assessment using assessmentId

GET /assessments/{assessmentId}

Generate Report

Create a report from a completed or partial assessment

POST /reports
Best practice: use the auto-filled completedAt from the fetched assessment instead of typing it manually.

Get Report

Fetch a report using reportId

GET /reports/{reportId}
